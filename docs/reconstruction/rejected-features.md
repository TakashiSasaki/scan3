# Rejected Features

この仕切り直しで初期実装から除外するもの（所有者の明示的判断なしに再導入しないもの）：

- legacy-to-new migration
- import protocol
- dual-write
- backfill
- reconciliation
- runtime compatibility layer
- EFP callable-functions API
- fact command receipt
- request hash/idempotency runtime
- Functions schema vendoring
- historical runtime profile
- historical application 2.x version lineage
- historical EFP 2.x/3.x version lineage
